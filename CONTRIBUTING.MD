# Contributing

Thank you for your interest in contributing to StudyQuiz.

Please follow these guidelines:

* Only issues with an approved contributor label are open to external contributions.
* Comment on the issue before starting work so that the scope can be confirmed.
* Create a separate branch for your changes and submit them through a pull request.
* Run and pass all relevant tests before submitting your pull request.
* Do not include secrets, credentials, production data, or private configuration.
* Clearly explain the changes made in the pull request.
* If AI-assisted tools were used, you must understand, review, and be able to explain the submitted code.
* Keep all changes within the agreed scope of the issue.
