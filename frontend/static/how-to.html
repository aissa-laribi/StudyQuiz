Generate Quizzes with ChatGPT

Copy the prompt, paste your slides into ChatGPT, and get a ready-to-upload JSON for StudyQuiz. No extra formatting required.

Quick Steps

  1. Open ChatGPT and have your slides/notes ready.
  2. Copy the prompt below and paste it into ChatGPT.
  3. Paste your slides right after the prompt.
  4. Ensure the reply is pure JSON (no markdown fences, no commentary).
  5. Copy the JSON into StudyQuiz → Import from ChatGPT → Upload Quiz.

Tip: If ChatGPT includes ```json fences or text, delete everything except the JSON object.

Copy‑Paste Prompt (strict)

This prompt tells ChatGPT to output only JSON and to conform to the StudyQuiz format.

Exact JSON Structure

StudyQuiz expects a top-level object with a questions array. Each item has a name (string) and an answers array of exactly 5 items with one correct: true.

{
  "questions": [
    {
      "name": "What does CPU stand for?",
      "answers": [
        {"name": "Central Processing Unit", "correct": true},
        {"name": "Computer Power Unit", "correct": false},
        {"name": "Core Programming Utility", "correct": false},
        {"name": "Central Performance Utility", "correct": false},
        {"name": "Compute Processing Utility", "correct": false}
      ]
    }
  ]
}

Note: No markdown fences, no trailing commas, and booleans are lowercase (true/false).

JSON Schema (for validators)

Link: /spec/quiz.schema.json (serve this as a raw JSON file). If you don’t have it yet, here’s the minimal schema you can copy into that file:

{
  "$schema": "https://json-schema.org/draft/2020-12/schema",
  "title": "StudyQuiz Quiz",
  "type": "object",
  "required": ["questions"],
  "properties": {
    "questions": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "object",
        "required": ["name", "answers"],
        "properties": {
          "name": { "type": "string", "minLength": 1 },
          "answers": {
            "type": "array",
            "minItems": 5,
            "maxItems": 5,
            "items": {
              "type": "object",
              "required": ["name", "correct"],
              "properties": {
                "name": { "type": "string", "minLength": 1 },
                "correct": { "type": "boolean" }
              },
              "additionalProperties": false
            }
          }
        },
        "additionalProperties": false
      }
    }
  },
  "additionalProperties": false
}

Common Issues & Quick Fixes

  • Markdown fences present → remove ```json lines.
  • More/less than 5 answers → adjust to exactly 5.
  • Multiple correct: true → set only one to true.
  • Invalid JSON → check commas, quotes, and braces.
  • Special characters → ensure they’re properly escaped (e.g., quotes).

Upload to StudyQuiz

  1. In StudyQuiz, choose Import from ChatGPT.
  2. Paste your JSON in the text area.
  3. Click Upload Quiz.

The importer validates the structure and will report any problems to fix.
